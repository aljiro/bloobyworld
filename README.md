
# Bloobyworld
Simulation of a simple agent that needs to be fed. The simulation and the dashboard of feeding are two separate applications that communicate using socket.io and nodejs.

# Dependencies
- Nodejs
- expressjs
- webpack
- socket.io
- processingjs

# Running
```
npm run server
```
